# Crypto-Time-Series-Analysis
A futuristic Bitcoin analytics dashboard and prediction engine. Features real-time price tracking, AI-powered forecasting (ARIMA, Prophet, LSTM), and sentiment analysis with a high-end Cyberpunk UI.

₿ CryptoTime Analytics Engine. CryptoTime Analytics is a full-stack cryptocurrency dashboard that blends data science with a modern web UI. This project not only tracks live Bitcoin statistics but also utilizes a multi-model AI approach to estimate future price trends.

✨ Key Features📊 Live Metrics Dashboard: Real-time data for current price, 24-hour change, market capitalization, and trading volume.

📈 AI-Powered Forecasting: Future predictions using three powerful algorithms:

ARIMA: Statistical Time Series analysis.Prophet: A trend and seasonality model developed by Meta (Facebook).

LSTM: A deep learning-based Long Short-Term Memory network (Simulated).

🧠 Sentiment Analysis: Utilizes Natural Language Processing (NLP) to detect 'Bullish' or 'Bearish' sentiment based on news or tweets.

🔍 Archive Data Search: Search for Bitcoin historical data from any past date using the Yahoo Finance API.

🎨 Cyberpunk UI: An animated, neon-themed user interface featuring custom CSS injections within Streamlit.

🛠️ Tech Stack:
Frontend: Streamlit, Plotly, CSS3 Animations.

Backend: FastAPI (Python).

Data Analysis: Pandas, NumPy.

AI/ML: Statsmodels (ARIMA), Prophet, TextBlob (NLP).

API/Data: Yahoo Finance (yfinance).
